Version Control System (VCS): sistem yang menyimpan & mengelola rekaman perubahan dari source code
Git: software atau aplikasi yang bisa melakukan VCS atau bisa juga disebut alat yang mengelola perubahan kode di komputer secara lokal
GitHub: website/layanan berbasis cloud untuk mengelola project Git

Git bisa untuk menghubungkan GitHub dengan komputer kita. Dengan syarat:
>> Buat GitHub menjadi Remote (sumber dari repo di github), lalu repo nya di clone ke komputer. Baru bisa push dan pull

Untuk melihat commit dalam bentuk titik horizontal secara visual: 
- Klik Insight (pada menu yang ada di atas)
- Klik Network (yang ada di sebelah kiri)

Alasan tidak bisa merge: 
- Ada konflik
Cara: 
  > Jalankan merge: git merge nama-branch
  > Kalau muncul konflik, buka file yang ditandai <<<<<<<, =======, >>>>>>>.
  > Edit manual pilih versi yang benar, lalu: 
    git add <file_yang_selesai_diedit>
    git commit
- Branch belum di-commit
Cara: 
  > Simpan dulu dengan commit atau stash:
    git add .
    git commit -m "Save progress"
  > Setelah itu lakukan merge
- History branch tidak sinkron
Cara:
  > Update dulu branch tujuan (misalnya main):
    git checkout main
    git pull origin main
  > Lakukan merge
- Protection rules (di GitHub/GitLab)
Cara:
  > Merge lewat Pull Request (PR).
  > Atau minta izin admin repo untuk merge langsung.
